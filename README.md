A minimal Python example project that prints **Hello, World!**.

---

## 📦 Requirements

- Python 3.8+

Check your version:

```bash
python3 --version
```

🚀 Run

Clone repository:

```bash
git clone https://github.com/yekki1n/hello-world-python.git
cd hello-world-python
```

Run program:

```bash
python3 hello.py
```

Output:

```
Hello, World!
```
